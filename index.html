Om oss

OM PUCKMARKET

Sveriges första marknadsplats byggd specifikt för ishockeyutrustning. 5% provision — transparent och enkelt.

Vår vision

Varje hockeyspelare ska ha tillgång till bra utrustning. Genom att ge begagnad och pro stock-utrustning nytt liv gör vi hockey billigare och mer hållbart.

Affärsmodell

PuckMarket tar 5% provision på varje genomförd försäljning. Säljaren behåller 95%. Inga dolda avgifter, inga abonnemang.

Köpguide

SÅ KÖPER DU

Steg-för-steg guide för köp på PuckMarket.

1. Hitta utrustning

Sök, filtrera på kategori, stad, skick, pris och typ (privat/förening/pro stock).

2. Kontakta säljaren

Skicka meddelande direkt via plattformen. Fråga om skick, storlek eller frakt.

3. Betala

💳 Swish — Rekommenderat

Snabbt och säkert. PuckMarket tar 5% provision som dras från säljarens del.

4. Frakt

📦 PostNord

1-3 vardagar. Spårningsnummer.

📦 Instabox

Snabb leverans till skåp.

🚗 Hämta

Möt säljaren på plats.

🚚 Bussgods

Bra för tung utrustning.

5. Betygsätt

Lämna betyg efter köpet för att bygga förtroende.

FAQ

VANLIGA FRÅGOR

Vad kostar det?

5% provision per försäljning. Gratis att lägga upp annonser.

Hur betalar jag?

Direkt mellan köpare och säljare, oftast Swish. Provisionen hanteras vid orderbekräftelse.

Vad är Pro Stock?

Utrustning använd av spelare i föreningar. Märks med orange badge.

Hur funkar massuppladdning?

Föreningar kan importera CSV-filer med många produkter samtidigt.

Hur registrerar jag förening?

Skapa konto → Klicka "Registrera förening" → Fyll i info → Klart.

Kan jag ta bort annons?

Ja, via "Mina annonser".
